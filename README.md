# echo-example

Example application of Echo (Go web framework). This use validator and translator.

## Usage

Set $DSN for PostgreSQL connection string.

## Installation

```
$ git clone https://github.com/mattn/echo-example
$ cd echo-example
$ go get -d
$ go build
```
